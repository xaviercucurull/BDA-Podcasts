apple_podcasts_scraper.get_podcasts_from_genre(genre_url, genre='', debug=False)¶

Given a genre url, obtain the title of all the podcasts.

Iterates over all the pages corresponding to each of the possible letters and find the title of all podcasts in each page.

Parameters

  • genre_url (str) – url of the genre podcast page.

  • genre (str, optional) – Name of the genre. Defaults to “”.

  • debug (bool, optional) – Print debug information. Defaults to False.

Returns

containing titles of podcasts

Return type

list
