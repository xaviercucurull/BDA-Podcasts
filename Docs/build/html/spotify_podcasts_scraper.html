SpotifyScraper¶

Spotify Podcasts Scraper

Class that uses the Spotipy package, which wraps the Spotify Web API. Can be used to extract shows and episodes from Spotify.

Author: Xavier Cucurull Salamero <xavier.cucurull@estudiantat.upc.edu> Course: 2021/2022

class spotify_podcasts_scraper.SpotifyScraper(client_id, client_secret)¶

Bases: object

Implementation of a Spotify Podcasts Scraper using Spotipy.

Get a list of shows from a suggested show_name and search for episodes to obtain further information such as show release date, last episode date and average duration of the episodes.

Example

>>> sp = SpotifyScraper(SPOTIPY_CLIENT_ID, SPOTIPY_CLIENT_SECRET)
>>> show = sp.search_shows('Gent de Merda')[0]
>>> episodes = sp.get_show_episodes(show)

search_shows(show_name, limit=10)¶
